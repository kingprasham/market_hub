# Spot Price Enhancements - Documentation

## Overview
This documentation covers the latest enhancements to the Spot Price tab, including visual grouping headers and change-based timestamps.

## New Features
### 1. Metal Category Headers
- Promptly identifies and displays metal categories (e.g., COPPER, BRASS, ALUMINIUM) from the Google Sheet.
- Styled with a vibrant yellow background to match the sheet's design.

### 2. Last Updated Timestamps
- Real-time tracking of price changes.
- Displays "Updated: [Relative Time] ago" for each item.
- Applies to Non-Ferrous, Steel, and Minor metal categories.

### 3. Navigation Improvements
- New Non-Ferrous detail route for deeper data analysis.
- Cleaned up sub-category filters for better usability.
